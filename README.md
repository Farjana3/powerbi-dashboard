# 📊 Power BI Sales Dashboard Project

## 📌 Overview
This project is an end-to-end Power BI dashboard built using a sales dataset. It demonstrates data cleaning, transformation, and visualization techniques to generate business insights.

The dashboard helps stakeholders understand sales performance, profitability, and trends through interactive visuals.

---

## 🚀 Features
- Data Import from dataset
- Data Cleaning & Transformation using Power Query
- Data Modeling
- DAX Measures & Calculations
- Interactive Dashboard Design
- KPI Cards (Sales, Profit, Margin)
- Sales Trend Analysis (Date-wise)
- Sales by Category and Region
- Top 5 Products by Sales
- Filters and Slicers (Date, Region, Category)

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX (Data Analysis Expressions)

---

## 📂 Dataset
- Sample Superstore Sales Dataset (or your dataset name)

---

## 📊 Dashboard Insights
- Identified top-performing products and regions
- Analyzed sales trends over time
- Compared category-wise performance
- Evaluated profitability across different segments
